# Project-Ark
본 프로젝트는 대한민국 자연인 최현의 꿈이자 남은 삶에서 이루고자 하는 목표이며,
즐거운 삶을 위한 최소한을 이루려는 노력이다.
거대한 범주에서 생물이자 자연인 한 사람으로서 '에너지 자립' 이라는 아이디어를 구체적인 프로젝트로 청사진을 그려 실행하고 그 과정에서 나오는 노하우와 방법을 오픈소스로 게시하려는 것에 목적이 있다.
